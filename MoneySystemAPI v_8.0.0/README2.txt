MoneySystemAPI 最新バージョン 5.0.0

=+=+=+=+=MoneySystemAPI=+=+=+=+=

・バージョン 5.0.0

・説明: サーバーに新しい経済要素のMoneySystemを導入します。

・コマンド一覧:

/mymoney : 自分の所持金を取得し、表示します。

/pay <対象プレイヤー名> <金額> : 自分の所持金を、他人に渡します。

/see <対象プレイヤー名> : 他人の所持金を確認します。

/mystatus : 自分の所持金がサーバー全体の何割を占めているかをパーセンテージで表示します。

/addmoney <対象プレイヤー名> <金額> : 指定したプレイヤーの所持金を増やします。 (OP限定コマンド)

/setmoney <対象プレイヤー名> <金額> : 指定したプレイヤーの所持金を設定します。 (OP限定コマンド)

/take <対象プレイヤー名> <金額> : 指定したプレイヤーの所持金を減らします。 (OP限定コマンド)

・ライセンス:

・このプラグインは Minecraft PocketEdition のマルチプレイサーバー「PocketMine-MP」を介してマルチプレイをする場合の追加要素(経済)を配布する目的で作成されました。

=+=+=+=+=MoneySystemLand=+=+=+=+=

・バージョン 1.6.9

・説明: 経済要素プラグインのMoneySystemに土地保護機能を追加します。

・コマンド一覧:

/startp : 今立っている場所を土地指定位置(一番)に設定します。

/endp : 今立っている場所を土地指定位置(二番)に設定します。

/land buy : 現在進行中の土地購入手続きを終了し、土地を購入します。

/buystop : 現在進行中の土地購入手続きを終了します。

/land list <ページ> : サーバーに存在する土地を表示します。

/land give <土地番号> <対象プレイヤー名> : 土地を譲渡します。

/land sell <土地番号> : 土地を売却します。

/land mark <土地番号> : 土地をマークアップします。

/land unmark <土地番号> : 土地のマークアップを解除します。

/land invite <土地番号> <対象プレイヤー名> : 土地共有をします。

/land unvite <土地番号> <対象プレイヤー名> : 土地共有を解除します。

/land here : 立っている場所の土地を検索します。

/land tp <土地番号> : 土地にテレポートします。

/land help : ヘルプを表示します。

/land info <対象プレイヤー名> : プレイヤーの土地情報を表示します。

/land invitee <土地番号> : 土地共有者リストを表示します。

・ライセンス:

・二次配布は禁止です。
・改造配布は禁止です。
・製作者の偽りは禁止です。
